# vscode相关知识

# 自定义代码片段

在vscode中，新建html文件，输入英文'!'按回车即可生成html默认代码
我们也可以自定义代码片段，步骤如下

  1. 文件->首选项->用户片段->新建全局代码片段文件
  2. 输入自定义片段名称，回车
  3. 按照以下格式添加即可(示例vue3)
{
    "vue3": {
        "prefix": "vue3",               //唤醒代码块的命令
        "body": [                       //body里就是生成的代码块，每一行都要用双引号包起来
            "<template>",
			"	<div>",
			"		$0",                //$0定义的是鼠标悬停的位置
			"	</div>",
			"</template>",
			"	",
			"<script lang='ts'>",
			"	export default {",
			"		name: '',",
			"		setup() {",
			"			",
			"			return{",
			"			}",
			"		},",
			"	};",
			"</script>"
        ],                              //代码块的主体
        "description": "vue3基础代码块"   //代码块的描述
    }
}
